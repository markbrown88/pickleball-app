## Recommended Statistics to Display

### 1. **Individual Player Rankings**

#### A. Overall Performance
- **Win Percentage** (Primary ranking metric)
  - Overall win/loss record
  - Minimum games threshold (e.g., 5+ games for ranking)
  - Example: Amna Jamal (90.0% - 9W-1L)

- **Games Played** (Participation metric)
  - Shows activity level
  - Useful for filtering "most active" vs "most successful"
  - Example: Ryan Bilodeau (38 games)

- **Point Differential**
  - Total points scored vs points allowed
  - Shows dominance in wins and competitiveness in losses
  - Formula: `Points For - Points Against`

#### B. Performance by Game Type
- **By Slot Breakdown**
  - Men's Doubles win percentage
  - Women's Doubles win percentage
  - Mixed 1 win percentage
  - Mixed 2 win percentage
  - Shows versatility and strengths

- **Format-Specific Records**
  - Singles (if applicable)
  - Doubles (combined men's + women's)
  - Mixed (combined mixed 1 + mixed 2)

#### C. Performance by Tournament Segment
- **By Bracket** (Skill level)
  - Performance in each bracket (e.g., 3.0, 3.5, 4.0+)
  - Shows consistency across skill levels

- **By Stop** (Location/Date)
  - Performance at each tournament stop
  - Shows consistency over time/location

#### D. Advanced Metrics
- **Clutch Performance**
  - Win percentage in deciding games (final game of match)
  - Win percentage in close games (margin â‰¤ 2 points)
  - Shows ability to perform under pressure

- **Margin of Victory/Defeat**
  - Average margin in wins
  - Average margin in losses
  - Shows dominance or competitiveness

- **Streak Statistics**
  - Current win/loss streak
  - Longest win streak
  - Longest loss streak

### 2. **Pair/Partnership Statistics**

#### A. Pair Performance
- **Pair Win Percentage**
  - Overall record when playing together
  - Minimum games threshold (e.g., 3+ games together)
  - Example: Drew Carrick & Spencer Carrick (100% - 16W-0L)

- **Pair Games Played**
  - Frequency of partnership
  - Shows most common pairings

- **Pair Performance by Slot**
  - How well the pair performs in each game type
  - Identifies best format for the partnership

#### B. Partner Analysis
- **Best Partners** (for each player)
  - Win percentage with each different partner
  - Most successful partnership combinations
  - Number of games with each partner

- **Partner Versatility**
  - Number of unique partners
  - Win percentage with different partners
  - Shows adaptability

### 3. **Team/Club Statistics**

#### A. Team Performance
- **Team Win Percentage**
  - Overall team record
  - Team point differential

- **Team Contribution**
  - Individual player's contribution to team wins
  - Games won vs games lost for team

#### B. Bracket Performance
- **Performance by Skill Bracket**
  - How players/pairs perform in different brackets
  - Shows if players are appropriately placed

### 4. **Comparative Statistics**

#### A. Rankings
- **Top 10/20 Players by Win Percentage** (min games threshold)
- **Top 10/20 Most Active Players**
- **Top 10/20 Players by Point Differential**
- **Top 10/20 Pairs by Win Percentage**
- **Top 10/20 Most Active Pairs**

#### B. Percentiles
- **Player Percentile Rankings**
  - Where each player ranks (e.g., "Top 10%", "Top 25%")
  - Based on win percentage, games played, etc.

### 5. **Trend Analysis**

#### A. Performance Over Time
- **Recent Form**
  - Last 10 games win percentage
  - Rolling 30-day performance
  - Shows if player is improving or declining

#### B. Consistency Metrics
- **Standard Deviation of Performance**
  - Consistency of win percentage across stops/brackets
  - Lower deviation = more consistent

### 6. **Special Achievements**

#### A. Milestones
- **Perfect Records** (100% win rate with minimum games)
- **Undefeated Pairs** (e.g., Drew Carrick & Spencer Carrick: 16-0)
- **Most Games Played**
- **Most Wins**
- **Longest Win Streak**

#### B. Notable Performances
- **Highest Win Percentage** (with meaningful sample size)
- **Most Dominant Victories** (largest point differentials)
- **Best Comeback Performances** (wins after being down)
